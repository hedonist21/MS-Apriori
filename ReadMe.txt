Follow the below steps to successfully execute the implementation of MS Apriori algorithm in python-

1-Open the file MSApriori.py
2-Update the path of parameter file in 'parameter_file' variable.
3-Update the path of file containing transaction in 'transaction_file' variable.
4-Execute the file MSApriori.py on your python shell.
5-The result printed on the shell is also stored in the result file.
6-The path of this file has to be specified in the variable 'output_file'.
